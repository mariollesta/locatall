# Locatall

Locatall is a web application designed to help you find the best places nearby based on your location and preferences.

- üìè **Distance Selection**: Search for locations in customisable radii (1 km, 5 km or 15 km). 
- üìä **Sorted by Popularity**: Check the best rated places in the area according to Google Places score.
